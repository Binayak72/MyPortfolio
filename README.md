# Binayak Kumar Mahato — Portfolio

A sleek, dark-themed developer portfolio built with Node.js and Express.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run the server
npm start
```

Visit: http://localhost:3000

## 🛠️ Tech Stack

- **Server**: Node.js + Express
- **Frontend**: Vanilla HTML/CSS/JS
- **Fonts**: Syne (display) + JetBrains Mono
- **Design**: Dark theme with cyan accent, terminal aesthetic

## 📁 Structure

```
portfolio/
├── server.js          # Express server
├── package.json
├── views/
│   └── index.html     # Main portfolio page
└── public/
    ├── css/
    │   └── style.css  # All styles
    └── js/
        └── main.js    # Interactions & animations
```

## ✨ Features

- Custom cursor with follower
- Typewriter title animation
- Animated skill cards with intersection observer
- Terminal-style about card
- Smooth scroll with active nav highlighting
- Floating particle effect
- Counter animation for stats
- Fully responsive
